Welcome to Neuroshima Hex PC's documentation!
=============================================

Neuroshima Hex PC is a computer version of famous board game Neuroshima HEX. This is a game of tactics, where armies wage continuous battles against each other. It is based on a roleplaying game called Neuroshima RPG published by Portal in 2001.
Neuroshima Hex PC is developing project based on `PyGame <http://pygame.org>`_. At this moment in project there are 3 armies: `Borgo`, `Moloch` and `Hegemony`. This game can be played by 2 players one against other in hotseat mode. Further work includes adding 4th army (`Outpost`), making convinient main menu, adding modes for 3 and 4 players.

Rules of Neuroshima Hex can be found `here <http://zmangames.com/rulebooks/Neuroshima_Hex_3.pdf>`_.
